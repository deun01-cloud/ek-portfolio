# EK PORTFOLIO

자산현황 추적부터 매수/매도 의사결정 지원까지 — 단일 HTML 파일로 동작하는 개인 자산관리 도구.

> 🚀 **처음 사용하시나요?** 먼저 [SETUP.md](./SETUP.md)로 셋업을 진행하세요.

---

## 📋 주요 기능

### 🏠 대시보드
- 사람별 탭 (본인/자녀1/자녀2/배우자/회사) — 한 클릭으로 그 사람 자산만 보기
- 총자산 / 1·7·30일 증감률 (KPI 카드 + 스파크라인)
- **자산 추이 그래프** — 1M / 3M / 6M / 1Y / 전체 기간 토글
- 분포 도넛: 계좌 종류별 / 계좌별 / 산업별
- 자산 배분 게이지 (목표 vs 실제)
- 벤치마크 비교 (KOSPI vs S&P500 vs 내 포트폴리오)
- **알림 패널** (접이식): 매수/매도/리밸런싱 신호 한 줄 요약 → 펼쳐서 개별 확인

### 🏦 계좌 관리
- 본인/자녀/회사/배우자/기타 계좌 관리
- **소유자명 필드**로 사람별 분리 (대시보드 탭 자동 생성)

### 📦 포트폴리오
- 위쪽: ⚖ 자산 배분 룰 (산업/종목/계좌별 비중 추적)
- 아래쪽: 보유 종목 표 (목표가/손절가/매수 논리)

### 🔍 퀀트 분석 + ★ 관심종목 통합
- 위쪽: 관심종목 카드 그리드 (F-Score 순) — 3가지 상태(검토/모니터/관심) + 목표 매수가
- 아래쪽: 종목 분석 (PER, PBR, ROE, 부채비율, 영업이익률 + Piotroski F-Score 9점)
- 3가지 비교 (산업 / 사용자 지정 / 동시 비교)
- 손익비 시뮬레이션

### ⇄ 거래내역
- 매수/매도 기록 + 매매 논리 메모
- **세후 실현손익 자동 계산** (한국/미국 양도세 + 수수료)
- **📥 증권사 CSV 임포트** — NH/대신/한투 등 자동 컬럼 인식
  - 거래내역 자동 추가 (중복 제외)
  - 보유 종목 자동 재계산
  - 예수금 자동 갱신

### ⚙ 설정
- API 프록시 URL
- 세율 / 수수료 / 환율
- 벤치마크 티커
- 데이터 내보내기/가져오기 (JSON)

## 📊 티커 형식

| 거래소 | 형식 | 예시 |
|---|---|---|
| KOSPI | `종목코드.KS` | `005930.KS` (삼성전자) |
| KOSDAQ | `종목코드.KQ` | `091990.KQ` (셀트리온헬스케어) |
| 뉴욕 / 나스닥 | 그대로 | `AAPL`, `TSLA`, `MSFT` |
| 지수 | `^심볼` | `^KS11` (KOSPI), `^GSPC` (S&P500) |

CSV 임포트 시 종목 코드는 자동 정규화됩니다 (6자리 숫자는 자동 KOSPI/KOSDAQ 분류).

## 💡 사용 팁

**사람별 자산 관리**
- 같은 소유자명을 가진 계좌들은 대시보드에서 한 탭에 합산
- 예: "키움 본인계좌" + "NH 본인계좌"의 소유자명 둘 다 "나" → 본인 탭에 합산

**매도 신호 활용**
- 보유 종목에 목표가/손절가 설정 → 시세 갱신 시 자동 알림
- 알림은 한 번 닫으면 안 뜨지만, 가격이 다시 변하면 새 알림으로 재등장

**매수 신호 활용**
- 관심종목 "🟡 모니터링" 상태 + 목표 매수가 설정
- 시세가 그 가격 이하로 떨어지면 대시보드 알림

**퀀트 분석 정확도 향상**
- 같은 산업 종목들을 여러 번 분석하면 산업 평균이 점점 정확해짐
- 예: 삼성전자 → SK하이닉스 → 마이크론 분석 → "Semiconductors 산업 평균"이 의미 있게 표시

**CSV 임포트 활용**
- 매번 거래 입력보다 월 1~2회 CSV 임포트가 훨씬 편함
- 같은 CSV를 여러 번 올려도 안전 (중복 자동 처리)

**자산 배분 룰 가이드**
- 단일 종목 15% 이내 / 단일 산업 30% 이내
- 자산군 예시: 한국주식 40% / 미국주식 40% / 현금 20%

## 🔒 데이터 안전성

- **모든 데이터는 브라우저 localStorage**에 저장 (서버 전송 ❌)
- Cloudflare Worker는 시세/지표 조회만 (개인 데이터 송신 ❌)
- 백업: 설정 → JSON 내보내기 (다른 기기에서 가져오기 가능)
- 증권사 비밀번호/API 키 입력 ❌ (CSV 다운로드 후 임포트 방식)

## 🛠 기술 스택

- **Frontend**: 단일 HTML (vanilla JS, ~4,200 라인)
- **차트**: 직접 구현한 SVG (외부 차트 라이브러리 없음)
- **CSV 파싱**: SheetJS (CDN)
- **시세/지표**: Yahoo Finance API (Cloudflare Worker 경유)
- **저장**: 브라우저 localStorage
- **호스팅**: GitHub Pages 또는 로컬
